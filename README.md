# Simple Win10 debloater

Um conjunto de scripts para eliminar debloatware e maximizar performance no Windows 10.

## Como executar os scripts?
Para utilizar, basta executar em modo administrador cada arquivo .reg e .ps1 individualmente.

## ATENÇÃO

**Seja um dev responsável e utilizem os scripts por conta própria, não serei responsável por qualquer problema que possa acontecer em seu sistema operacional ao utilizarem esses scripts.**

Os scripts são baseados no repositório: https://github.com/Sycnex/Windows10Debloater pelo qual selecionei e modifiquei alguns de seus scripts para criar essa versão lite contendo apenas modificações que julguei úteis para configuração do Windows 10 em minha rotina, super recomendo que leiam todos os códigos .reg e .ps1 antes de executarem e se necessário modifiquem ao seu gosto.
